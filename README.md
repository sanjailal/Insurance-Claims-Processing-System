# Insurance Claims Processing System

A claims adjudication system built as a take-home assignment for Realfast.

---

## What It Does

- Accepts claim submissions with line items from members
- Adjudicates each line item against policy coverage rules
- Tracks claims and line items through their lifecycle
- Produces explanations for every coverage decision
- Supports member disputes with manual insurer override
